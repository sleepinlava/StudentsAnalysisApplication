# 相关自动化脚本的一些说明
___
 
## 关于下载
1. 方法一，使用shell命令行下载
   - `git clone https://github.com/sleepinlava/StudentsAnalysisApplication.git`
2. 方法二，在项目页下载zip文件，解压到本地后进行使用
___

## 关于使用
1. 本项目./data文件夹中内置了项目参考样表，请将表头改成相应的格式才可以自动化脚本
2. 本项目基于Python语言开发，同时借助PyInstaller进行打包，速度可能存在缓慢问题，请谅解
______
